# PBus2AMCP

PBus to AMCP translator.

## Getting Started

### Prerequisites

* Qt5
* QtSerialPort

### Installation

```
git clone https://github.com/dimitry-ishenko-casparcg/pbus2amcp.git
cd pbus2amcp
mkdir out
cd out
cmake ..
make
```

### Usage

```
./pbus2amcp
```
